1.1 Кіріспе. Python

Python — бұл бағдарламалау тілі, ол әртүрлі бағдарламалар, сайттар, ойындар және жасанды интеллект жүйелерін жасау үшін қолданылады. Python 1991 жылы жасалған. Python-ды Guido van Rossum ойлап тапқан. Көбінесе оны арнайы бағдарламаларда жазады, мысалы PyCharm сияқты ортада жұмыс істейді. Python арқылы адамдар веб-сайттар жасайды, деректерді талдайды, боттар құрады және әртүрлі процестерді автоматтандырады. Ол қарапайым синтаксисінің арқасында жаңадан бастаушыларға түсінікті және тез үйренуге ыңғайлы.
